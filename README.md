# Converter Pro (SVG to PNG/ICO)

Un'applicazione leggera in Python per convertire file SVG in PNG o ICO (icone Windows) mantenendo la massima qualità grazie all'integrazione con **Inkscape**.

## Funzionalità
- Conversione singola o di intere cartelle.
- Supporto per PNG e ICO (multi-risoluzione).
- Interfaccia Dark ottimizzata per alta leggibilità.
- Apertura automatica della cartella di destinazione.

## Requisiti
Per funzionare correttamente, il sistema deve avere installato:
- [Inkscape](https://inkscape.org/)
- [ImageMagick](https://imagemagick.org/) (per la generazione dei file .ico)
